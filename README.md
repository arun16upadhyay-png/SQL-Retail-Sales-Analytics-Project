# Retail Sales Analytics Project

## Project Overview
This project analyzes retail sales data using MySQL to generate business insights related to sales performance, profitability, customer behavior, returns, shipping efficiency, and customer retention.

## Tools Used
- MySQL
- SQL
- Data Analysis
- Data Cleaning
- Window Functions
- CTEs

## Dataset
- sales.csv
- calender_file.csv

## Business Questions Solved
1. Year-over-Year Sales & Profit Growth
2. Weekend Sales Analysis
3. Best Performing Quarter
4. Discount vs Profit Analysis
5. Regional Return Rate Analysis
6. Loss-Making Customer Identification
7. Shipping Delay Analysis
8. Late Delivery Impact on Returns
9. Salesperson Performance Evaluation
10. Pareto 80/20 Analysis
11. Customer Churn Analysis
12. 30-Day Moving Average Analysis

## SQL Skills Demonstrated
- Joins

- CASE Statements
- CTEs
- Window Functions
- Date Functions
- Business KPI Analysis
